# Smart Shipyard AI Safety - Frontend Mockup

요구사항 정의서를 기반으로 만든 React 프론트엔드 목업입니다.

## 실행 방법

Node.js 18 이상과 pnpm이 필요합니다.

```powershell
pnpm install
pnpm dev
```

터미널에 표시되는 `http://localhost:5173` 주소를 브라우저에서 엽니다.

## 화면 구성

- 로그인 및 역할 선택: 현장 작업자 / 관리자
- 현장 작업자: 오늘의 작업, TBM 브리핑, 조치 체크리스트, 위험 신고
- 관리자: 통합 관제, 영상 감시, 허가서 분석, 위험 예측, 기준 정보, 감사 로그

샘플 사번과 비밀번호가 미리 입력되어 있으며, 두 역할 중 하나를 선택한 뒤 바로 접속할 수 있습니다.

## 참고

이 결과물은 프론트 목업입니다. AI 모델, 음성인식, 번역, 알림, 서버, DB는 샘플 데이터와 UI 상호작용으로 표현되어 있습니다.
